Hallo Semuanya....

Perkenalkan nama saya Ririn Angraini, saya mahasiswa angkatan 2020 dari Prodi S1 Teknik Informatika, Jurusan Teknik Elektro, Fakultas Teknik Universitas Riau. Saya berasal dari desa Sungai Raya, Kec. Rengat, Kabupaten Indragiri Hulu, dan merantau ke Kota Pekanbaru untuk melanjutkan pendidikan saya di tingkat Perguruan Tinggi. Saya telah berkuliah selama 3 semester dan sekarang sedang menjalani perkuliahan semester 4. Saya sangat menikmati dunia perkuliahan karena bertemu dengan orang-orang baru serta belajar tentang hal-hal baru yang belum pernah saya temukan sebelumnya. Saya kurang menyukai hal - hal yang harus berbicara di depan umum, dan saya menyukai hal - hal yang tidak terlalu menampakkan diri tetapi saya suka di belakang layar dan banyak bekerja.

Berikut ini adalah beberapa alasan saya memilih Prodi Teknik Informatika sebagai Jurusan Kuliah Saya :

  3. Bisa kerja dimana saja.
